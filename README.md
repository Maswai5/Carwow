# AutoTrade (Carwow Clone)

AutoTrade is a modern web application for buying and selling vehicles, inspired by Carwow. Built with React, TypeScript, Vite, Tailwind CSS, and Shadcn UI.

## Features
- Browse thousands of cars from trusted sellers
- Post vehicle listings
- Responsive design
- Accessible UI components
- Fast development with Vite
- Robust testing with Vitest, Playwright, and Testing Library

## Getting Started

### Prerequisites
- Node.js (v18+ recommended)
- npm (or bun/yarn)

### Installation
```bash
npm install
```

### Running the Development Server
```bash
npm run dev
```
Visit [http://localhost:8080](http://localhost:8080) in your browser.

### Building for Production
```bash
npm run build
```

### Running Tests
```bash
npm run test
```

## Project Structure
- `src/` — Main source code
- `src/components/` — UI and app components
- `src/pages/` — Page components
- `src/data/` — Vehicle data
- `src/hooks/` — Custom React hooks
- `src/lib/` — Utility functions
- `public/` — Static assets

## Usage Example
To add a new vehicle listing, click the "Post Listing" button in the navbar and fill out the form.

## Customization
- Update `tailwind.config.ts` for custom colors and themes
- Add new pages/components as needed
